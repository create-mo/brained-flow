# wiki-setup — Claude Cowork Skill

Скилл для установки и настройки системы двусторонней синхронизации `brain/` ↔ claude.ai Projects.

## Что делает

Помогает Claude провести тебя через весь процесс настройки с нуля:
- Получить session key из браузера
- Узнать ORG_ID и project_id
- Настроить скрипты под свои пути и проекты
- Проверить что всё работает
- Настроить автозапуск wiki-watch при логине

## Связанные скиллы

- **brain-sync** — использование уже настроенной системы (push/pull/watch)
- **wiki-setup** (этот скилл) — первоначальная установка и настройка

## Требования

- Python 3.x
- Windows (wiki-watch.ps1 использует PowerShell и FileSystemWatcher)
- Аккаунт claude.ai
